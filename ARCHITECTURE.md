## Поток данных

```
УЧЕНИК (Браузер)
    │
    ├─ Пишет сообщение
    │
    ▼
VERCEL (Ваш сервер)
    │
    ├─ Получает сообщение
    ├─ Проверяет переменные окружения
    │
    ▼
GOOGLE GEMINI API
    │
    ├─ Получает запрос
    ├─ Обрабатывает (система: "ты Концепт Учитель")
    ├─ Генерирует ответ на основе интересов
    │
    ▼
VERCEL (Ваш сервер)
    │
    ├─ Получает ответ от AI
    │
    ▼
УЧЕНИК (Браузер)
    │
    ├─ Видит ответ
    ├─ Данные сохраняются локально
    ├─ Получает XP
    │
    ▼
КОНЕЦ
```

## Развертывание

```
ВАШ КОМПЬЮТЕР (Локальная разработка)
    │
    ├─ Код в VS Code
    │
    ├─ Git push
    │
    ▼
GITHUB РЕПОЗИТОРИЙ (thinkflow-ai)
    │
    ├─ Код хранится здесь
    │
    ├─ Vercel следит за изменениями
    │
    ▼
VERCEL (Автоматический Deploy)
    │
    ├─ Скачивает код
    ├─ Устанавливает зависимости (npm install)
    ├─ Собирает приложение (npm run build)
    ├─ Заливает на серверы
    │
    ▼
ЖИВОЙ САЙТ
    │
    ├─ Доступен всем в школе
    ├─ Работает 24/7
    ├─ Автоматически обновляется при изменениях
    │
    ▼
✅ ГОТОВО!
```

## Безопасность

```
ПАРОЛИ:
    ├─ Хранятся в браузере (localStorage)
    ├─ Не отправляются на сервер ✅
    ├─ Видны только их обладателю ✅
    └─ Даже админ их не видит ✅

ДАННЫЕ:
    ├─ Все локально в браузере ✅
    ├─ История сообщений - в браузере ✅
    ├─ Профиль - в браузере ✅
    ├─ XP и уровень - в браузере ✅
    └─ Очистили браузер - данные удалены ✅

API КЛЮЧ:
    ├─ Хранится в Vercel (защищенные переменные) ✅
    ├─ Никогда не видим на клиентской стороне ✅
    ├─ Привязан только к Vercel домену ✅
    └─ Можно пересоздать в любой момент ✅
```

## Технический стек

```
FRONTEND:
├─ React 19
├─ TypeScript
├─ Tailwind CSS
├─ Vite (сборка)
└─ ESM (модули)

BACKEND/DEPLOY:
├─ Vercel (hosting)
├─ Node.js environment
└─ Edge Functions (опционально)

AI ENGINE:
├─ Google Gemini API
├─ @google/genai SDK
└─ Streaming responses

STORAGE:
├─ Browser localStorage
├─ (Опционально) Vercel KV database (будущее)
└─ (Опционально) Vercel Postgres (будущее)
```

## Производительность

```
ПЕРВАЯ ЗАГРУЗКА:
    ├─ Время: ~2-3 сек
    ├─ Размер: ~150KB (сжато)
    ├─ Почему долго: 
    │  ├─ Скачивается приложение
    │  ├─ Скачивается React
    │  ├─ Скачивается Tailwind CSS
    │  └─ Скачивается Google Gemini SDK
    └─ После - всё в кэше ✅

ПОСЛЕДУЮЩИЕ ЗАГРУЗКИ:
    ├─ Время: ~500ms
    └─ Всё из локального кэша ✅

ЗАПРОСЫ К AI:
    ├─ Время ответа: 3-10 сек
    ├─ Зависит от:
    │  ├─ Длины вашего вопроса
    │  ├─ Сложности запроса
    │  ├─ Нагрузки на Google API
    │  └─ Скорости интернета
    └─ Обычно очень быстро ⚡
```
