# 🎬 Визуальная инструкция по развертыванию

## Шаг 1️⃣: Получить API ключ (2 минуты)

```
┌─────────────────────────────────┐
│  Откройте в браузере:          │
│  https://aistudio.google.com/apikey
└─────────────────────────────────┘
         ↓
┌─────────────────────────────────┐
│  Вы увидите страницу            │
│  "Google AI Studio"             │
│  с кнопкой "Create API key"     │
└─────────────────────────────────┘
         ↓
┌─────────────────────────────────┐
│  Нажмите "Create API key"       │
│  Выберите проект                │
│  Копируем ключ:                 │
│  AIza...xyz (длинная строка)    │
└─────────────────────────────────┘
```

## Шаг 3️⃣: Импортировать проект (5 минут)

```

│  нажмите: "Add New..."          │
│           ↓                      │
│        "Project"                │
└─────────────────────────────────┘
         ↓
┌─────────────────────────────────┐
│  Выберите:                      │
│  "Import Git Repository"        │
│                                  │
│  Найдите: "thinkflow-ai"        │
└─────────────────────────────────┘
         ↓
┌─────────────────────────────────┐
│  Нажмите "Import"               │
└─────────────────────────────────┘
```
