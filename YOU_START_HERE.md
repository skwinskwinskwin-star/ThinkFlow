---

## 📋 ШАГ 1: Получить Google API ключ (2 минуты)

```
1. Откройте: https://aistudio.google.com/apikey
2. Нажмите кнопку "Create API key" (Создать API ключ)
3. Выберите проект (новый или существующий)
4. Скопируйте ключ - он выглядит как:
   AIza... (длинная строка)
5. Сохраните его где-нибудь (пригодится на шаге 3)
```

**Важно:** Не делитесь этим ключом с кем-либо!

---

## 🚀 ШАГ 2: Развернуть на Vercel (10 минут)

### 2.1: Создать аккаунт на Vercel

```
1. Откройте: https://vercel.com/signup
2. Нажмите "Continue with GitHub"
3. Авторизуйтесь в GitHub
4. Разрешите доступ Vercel
```

### 2.2: Импортировать ваш проект

```
1. На Vercel нажмите "Add New..." → "Project"
2. Нажмите "Import Git Repository"
3. Найдите "thinkflow-ai" в списке
4. Нажмите "Import"
```

### 2.3: Добавить API ключ

**В форме импорта найдите "Environment Variables":**

```
Name:  GEMINI_API_KEY
Value: [ваш ключ из шага 1]
```

Нажмите "Add"

### 2.4: Запустить Deploy

```
Нажмите кнопку "Deploy"
Ждите 2-3 минуты... ✅ Готово!
```

## 🆘 ЕСЛИ ЧТО-ТО НЕ РАБОТАЕТ

### Проблема: "API Key is invalid"
```
✓ Проверьте что ключ скопирован без пробелов
✓ Убедитесь что вы добавили его на Vercel
✓ Нажмите "Redeploy" в Vercel (Settings → Deployments)
✓ Подождите 2 минуты и обновите страницу
```

### Проблема: "Белый экран"
```
✓ Очистите браузер (Ctrl+Shift+Delete)
✓ Откройте в приватном окне (Ctrl+Shift+P)
✓ Попробуйте в другом браузере
✓ Обновите страницу (Ctrl+R)
```

### Проблема: "Медленно загружается"
```
✓ Это нормально при первом открытии
✓ Подождите 30 секунд
✓ Перезагрузите страницу (Ctrl+R)
```
