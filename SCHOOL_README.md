# 📚 ThinkFlow - Образовательная платформа вашей школы

## 🎯 Что это?

**ThinkFlow** - это персональный AI учитель, который объясняет темы через ваши интересы!

Забудьте о скучных определениях - мы объясняем физику через футбол, алгебру через видеоигры, английский через аниме!

## ✨ Возможности

### 1️⃣ Концепт Учитель 🎓
Объясняет любой предмет через ваши интересы
- Выберите интересы (Спорт, Аниме, Код, Бизнес...)
- Задайте вопрос
- Получите объяснение, которое вам понравится

### 2️⃣ Коуч Мышления 🧠
Учит вас находить ответы самостоятельно (Сократовский метод)
- Вместо ответа - вопросы
- Развивает логическое мышление
- Помогает понять - не запомнить

### 3️⃣ Адаптивный Тренер ⚡
Задачи, которые подстраиваются под ваш уровень
- Легкие задания для начинающих
- Сложные задачи для продвинутых
- Оценивает логику, а не просто правильность

## 🚀 Быстрый старт (30 секунд)

1. **Откройте ссылку:** https://[ваша-ссылка].vercel.app
2. **Нажмите "Get Started"** (Начать)
3. **Регистрируйтесь** или **Войдите**
4. **Выберите интересы** (минимум 1)
5. **Начните учиться!**
