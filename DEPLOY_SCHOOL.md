# 🚀 Как развернуть ThinkFlow для школы на Vercel

## За 5 минут - ваш сайт будет доступен всем!

### Шаг 1: Подготовка (1 минута)

Убедитесь что:
- ✅ Ваш код загружен в GitHub
- ✅ У вас есть Google аккаунт
- ✅ У вас есть email для Vercel (можно использовать GitHub аккаунт)

### Шаг 2: Получить API ключ (1 минута)

1. Перейдите: https://aistudio.google.com/apikey
2. Нажмите **"Create API key"**
3. Выберите проект (или создайте новый)
4. Скопируйте ключ (вам он понадобится на шаге 5)

### Шаг 3: Создать Vercel аккаунт (1 минута)

1. Откройте: https://vercel.com/signup
2. Нажмите **"Continue with GitHub"**
3. Авторизуйтесь в GitHub (если нужно)
4. Разрешите доступ Vercel к вашему GitHub

### Шаг 4: Импортировать проект (1 минута)

1. Вы вернётесь на Vercel
2. Нажмите **"Add New... → Project"**
3. Нажмите **"Import Git Repository"**
4. Найдите **thinkflow-ai** в списке
5. Нажмите **Import**

### Шаг 5: Добавить API ключ (1 минута)

После нажатия Import вы увидите форму:

**В разделе "Environment Variables":**
- **Name:** `GEMINI_API_KEY`
- **Value:** Вставьте ключ из шага 2

Нажмите **"Add"**

### Шаг 6: Deploy! (ENTER 🚀)

Нажмите кнопку **"Deploy"**

Ждите 2-3 минуты... готово! 🎉

## 🆘 Если что-то не работает:

**Ошибка "API_KEY not found"?**
- Проверьте что вы добавили переменную окружения на Vercel
- После добавления нужно пересоздать Deploy (Redeploy)

**Как пересоздать Deploy:**
1. Откройте проект на Vercel
2. Нажмите **"Deployments"**
3. Выберите последний deploy
4. Нажмите **"..." → "Redeploy"**
