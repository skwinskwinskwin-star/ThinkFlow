## 🚀 Как начать (4 простых шага):

### 1️⃣ Получить API ключ (2 мин)
```
Откройте: https://aistudio.google.com/apikey
Нажмите: "Create API key"
Скопируйте ключ
```

### 2️⃣ Создать Vercel аккаунт (3 мин)
```
Откройте: https://vercel.com/signup
Авторизуйтесь через GitHub
```

### 3️⃣ Развернуть проект (5 мин)
```
На Vercel: "Add New" → "Project"
Импортируйте: thinkflow-ai
Добавьте: GEMINI_API_KEY = [ваш ключ]
Нажмите: "Deploy"
```

### 4️⃣ Раздать ссылку школе (5 мин)
```
Скопируйте полученную ссылку
Отправьте в школьный чат/email
```
