# 👨‍💼 Инструкция для администратора школы

## 📋 Обзор

ThinkFlow - это AI-платформа обучения, которую вы можете развернуть для всех учащихся школы.

## ✅ Требования

- Google аккаунт (для API ключа)
- GitHub аккаунт (уже есть у разработчика)
- Интернет
- Всё! 🎉

## 🚀 Развертывание (15 минут, один раз)

### Этап 1: Получить API ключ (2 минуты)

1. Перейдите: https://aistudio.google.com/apikey
2. Нажмите **"Create API key"**
3. Выберите проект
4. **Скопируйте ключ** - он понадобится на этапе 3

### Этап 2: Создать Vercel проект (5 минут)

1. Откройте: https://vercel.com/signup
2. Авторизуйтесь через GitHub
3. Нажмите **"Add New... → Project"**
4. Импортируйте репозиторий `thinkflow-ai`

### Этап 3: Добавить API ключ (2 минуты)

В форме импорта:

```
Environment Variables:
Name:  GEMINI_API_KEY
Value: [ключ из этапа 1]
```

Нажмите **Deploy**

### Этап 4: Готово! (1 минута)

Вы получите ссылку вида:
```
https://thinkflow-ai.vercel.app
```

## 📢 Как раздать школьникам

1. **Скопируйте ссылку** из Vercel
2. **Отправьте в школьный чат** (Viber, Telegram, WhatsApp)
3. **Или добавьте на школьный сайт**
4. **Или отправьте email со ссылкой**

Школьники просто откроют ссылку и смогут начать!

## 🎮 Как это работает для учащихся

```
Школьник открывает ссылку
         ↓
Видит красивый интерфейс
         ↓
Нажимает "Get Started"
         ↓

         ↓
Выбирает интересы (Спорт, Аниме, Код...)
         ↓
Общается с AI учителем
         ↓
Получает объяснения через его интересы
```

## 🆘 Техническая поддержка

**Если что-то не работает:**

1. Проверьте что API ключ добавлен в Vercel
2. Нажмите "Redeploy" на последний deploy
3. Очистите браузер (Ctrl+Shift+Delete)
4. Попробуйте в приватном окне (InPrivate/Incognito)

**Основные проблемы:**
- "API не отвечает" → Проверьте ключ и квоту Google
- "Белый экран" → Очистите браузер и обновите
- "Медленно загружается" → Это нормально при первой загрузке
